Subject + want to be + Noun/Adj

ဒီ pattern လေးကတော့ "တစ်ခုခု ဖြစ်ချင်တယ်" လို့ ပြောချင်တဲ့အခါ သုံးလို့ရပါတယ်။ ရှေ့က pattern နဲ့ ဆင်ပေမဲ့ "be" လေး ပါလာတဲ့အတွက် အခြေအနေတစ်ခုခုကို ဖော်ပြတာ ဖြစ်သွားပါတယ်။

ရှင်းလင်းချက်

Noun: အသက်မွေးဝမ်းကျောင်း ဒါမှမဟုတ် လူပုဂ္ဂိုလ် (ဥပမာ - ဆရာဝန်)

Adj (Adjective): စိတ်ခံစားချက် ဒါမှမဟုတ် အခြေအနေ (ဥပမာ - ပျော်ရွှင်တာ)

Subject-Verb Agreement

I, You, We, They
+ want to be
He, She, It
+ wants to be

နမူနာစာကြောင်း (၅) ခု

1. I want to be a doctor.
(ကျွန်တော် ဆရာဝန်တစ်ယောက် ဖြစ်ချင်တယ်။)
2. I want to be happy.
(ကျွန်မ ပျော်ပျော်ရွှင်ရွှင် နေချင်တယ်။)
3. She wants to be a singer.
(သူမ အဆိုတော်တစ်ယောက် ဖြစ်ချင်တယ်။)
4. I want to be rich.
(ကျွန်တော် ချမ်းသာချင်တယ်။)
5. He wants to be a leader.
(သူ ခေါင်းဆောင်တစ်ယောက် ဖြစ်ချင်တယ်။)

🎯 အခုပဲ လေ့ကျင့်ကြည့်ရအောင်!

အောက်က စကားလုံးလေးတွေကို သုံးပြီး ဝါကျတည်ဆောက်ကြည့်ပါ။

1. Healthy (ကျန်းမာသော)
2. A Teacher (ဆရာတစ်ယောက်)
3. Successful (အောင်မြင်သော)
4. Brave (သတ္တိရှိသော)
5. An Engineer (အင်ဂျင်နီယာတစ်ယောက်)

ဥပမာ - "I want to be healthy."
(ကျွန်တော် ကျန်းမာချင်တယ်။)

အလွယ်မှတ်ချင်ရင်
